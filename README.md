# 🌑 Obsidian Command Center

**The Ultimate Design System for Home Assistant.**

![Dashboard Preview](dashboard-preview.png)

Welcome to the official repository for the Obsidian Command Center. This is a high-density, glassmorphic UI designed for those who want their smart home to feel like a pro-level OS.

## ✨ Key Highlights

- **Obsidian Theme**: Deep blacks, custom typography, and SVG noise textures.
- **Glassmorphic UI**: High-density room cards with neon accenting.
- **Interactive Controls**: Custom vacuum and media cards with CSS animations.
- **Complete Kit**: Includes Dashboard YAML, Theme, and Install Guide.

## 🚀 Quick Start

1. **Theme**: Copy [themes/obsidian.yaml](themes/obsidian.yaml) to your `/config/themes/` folder.
2. **Dashboard**: Use [dashboard/lovelace_dashboard_template.yaml](dashboard/lovelace_dashboard_template.yaml) as a template for your Lovelace UI.
3. **Guide**: Follow the [Step-by-Step Install Guide](INSTALL_GUIDE.md) for full setup instructions.

## 🛠️ Required HACS Integrations

Ensure you have these installed:

- `button-card`
- `mushroom`
- `mini-media-player`
- `layout-card`
- `card-mod`
